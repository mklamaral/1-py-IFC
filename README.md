#  Python @ IFC - Araquari

Este repositório contém meus exercícios, projetos e anotações das aulas de **Algoritmos e Programação** do curso de **Sistemas de Informação** no Instituto Federal Catarinense.

---

## Tecnologias e Ambiente
Para manter o desepenho no meu setup (Celeron + Lubuntu), utilizo:
* **Linguagem:** Python 3.12+
* **Editor:** VSCodium (Open Source VS Code)
* **OS:** Lubuntu 24.04 (Minimal Installation)

---

## Estrutura do Repositório
* `01-fundamentos/`: Variáveis, tipos de dados e operadores.
* `02-estruturas-decisao/`: If, else, elif.
* `03-estruturas-repeticao/`: While e For.
* `04-listas-dicionarios/`: Manipulação de dados e coleções.
* `projetos/`: Scripts mais complexos e desafios das aulas
